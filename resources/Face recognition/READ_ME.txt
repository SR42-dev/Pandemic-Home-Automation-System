To test this code with your own images ...

1. Upload your images into the 'Face' folder (intended to be empty initially). 
   You may rename the folder but if you do,ensure that you change the call to 
   the folder in dataset.py to the new name (commented)

2. Execute dataset.py (this will create a new file in the folder. you can ignore it but DO NOT delete it)

3. execute face_recog_live_feed.py and watch it recognize the encoded faces
